# HR Analytics Dashboard - Power BI

## 📊 Project Overview

This project is an HR Analytics Dashboard developed using Power BI to analyze workforce data and provide actionable insights. The dashboard helps organizations monitor employee demographics, salary distribution, performance ratings, experience levels, and department-wise workforce trends.

## 🎯 Objectives

* Analyze employee data effectively
* Track key HR metrics through KPI cards
* Identify workforce distribution across departments
* Monitor employee performance ratings
* Visualize salary and experience trends
* Support data-driven HR decision-making

## 📈 Key Performance Indicators (KPIs)

* Total Employees
* Average Salary
* Average Age
* Average Experience

## 📊 Dashboard Visualizations

* Department-wise Employee Distribution
* Gender Distribution Analysis
* Performance Rating Analysis
* Employee Count by Department
* Employee Details Table

## 🛠 Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* Microsoft Excel

## 📚 Skills Demonstrated

* Data Visualization
* Dashboard Design
* KPI Development
* DAX Measures
* Business Intelligence
* HR Data Analysis
* Reporting and Analytics

## 📂 Dataset Information

The dataset contains employee-related information such as:

* Employee ID
* Department
* Gender
* Age
* Salary
* Experience
* Performance Rating

## 🚀 Project Outcome

The dashboard provides a comprehensive view of HR metrics, enabling better workforce analysis and helping stakeholders make informed decisions based on employee data.
